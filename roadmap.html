Development Roadmap

Track ongoing development, upcoming features, and long-term vision for the TRANSPOSE ecosystem.

Q1 2025 Milestones

DCMBench

  • Release v1.0 with stabilized API and schema
  • Add support for nested logit and mixed logit models
  • Integrate automated data quality checks
  • Expand dataset collection with 3 new metropolitan regions

ML-DCM-Bench

  • Implement standardized cross-validation protocols
  • Add transformer-based baseline models
  • Create leaderboard system for model submissions
  • Release comprehensive evaluation metrics package

TransitCast

  • Real-time GTFS-RT integration
  • Multi-modal journey planning support
  • Weather and event impact modeling
  • Deploy demo instance with sample transit agency data

DNN-Econ

  • Welfare change calculation module
  • Cross-elasticity extraction for neural networks
  • Visualization toolkit for economic interpretations
  • Integration with SHAP and LIME for explainability

ASU-DNN

  • PyTorch Lightning refactor for better scalability
  • Distributed training support
  • AutoML integration for hyperparameter tuning
  • Pre-trained models for common choice contexts

Q2-Q3 2025 Priorities

Infrastructure

  • Unified CLI tool for all TDToolkit components
  • Cloud-based computation platform (AWS/GCP integration)
  • Automated benchmarking pipeline with nightly runs
  • Container orchestration for reproducible environments

Data & Privacy

  • Differential privacy implementation for sensitive datasets
  • Synthetic data generation toolkit
  • Federated learning framework for multi-agency collaboration
  • GDPR and CCPA compliance tools

New Components

  • ActivitySim-Bridge: Integration with activity-based models
  • TDViz: Interactive visualization library for choice models
  • CausalChoice: Causal inference toolkit for policy evaluation

Long-term Vision (2026+)

TRANSPOSE aims to become the standard platform for travel demand modeling research and practice, fostering collaboration between academia, public agencies, and industry partners. Key initiatives include:

  • Global expansion with region-specific models and datasets
  • Integration with smart city platforms and real-time data streams
  • Quantum computing experiments for large-scale optimization
  • Foundation models for transfer learning across cities
  • Automated model selection and ensemble methods
  • Climate impact assessment tools for transportation planning
